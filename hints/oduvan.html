Tem outro jeito de percorrer a lista de números?

Sim, vamos testar o loop for. Apesar do while também funcionar para esse exercício, o for é mais indicado para percorrermos sequências (como uma lista ou uma string) e executar comandos para cada item.

Ok, me dê um exemplo da sintaxe do for?

A variável 'numero' logo depois da palavra for representa cada item da lista, um de cada vez, em ordem crescente. Chamamos esse procedimento de iteração. 

 lista = [1, 2, 3, 4, 5]

for numero in lista:
    print(numero)

Perceba como não precisamos colocar uma condição e nem alterar o índice da lista quando usamos o loop for.

Como posso salvar a soma?

Não se esqueça de criar uma variável adicional para armazenar e atualizar o valor de soma para cada item da lista.

Ex: total = total + 1
